## 思路
从右到左遍历罗马数字的字符，将罗马字符映射为对应的阿拉伯数字，若当前的数字大于或等于前一个数字，则加，否则减。
例如：`XXVII`等于1+1+5+10+10 = 27 、`IX`等于10-1=9、`XXCI`等于1+100-10。

## 代码实现
```go
func RomanToInt(s string) (r int) {
	m := map[byte]int{
		'I': 1,
		'V': 5,
		'X': 10,
		'L': 50,
		'C': 100,
		'D': 500,
		'M': 1000,
	}
	pre := 0
	for i := len(s) - 1; i >= 0; i-- {
		cur := m[s[i]]
		if cur >= pre {
			r += cur
		} else {
			r -= cur
		}
		pre = cur
	}

	return r
}
```

## 复杂度
时间复杂度**O(n)**, 空间复杂度**O(1)**。
